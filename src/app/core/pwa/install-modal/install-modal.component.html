App installieren

So fügst du die App zum Home-Bildschirm hinzu:

  1. 1) Tippe auf das Teilen-Symbol
  2. 2) Wähle „Zum Home‑Bildschirm“
  3. 3) Tippe auf „Hinzufügen“
Hinweis: Diese Anleitung gilt für Safari auf iPhone/iPad. Auf Android wird dir bei Verfügbarkeit eine Installationsaufforderung angezeigt.
